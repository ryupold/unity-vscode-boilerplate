I created this project setup based on this tutorial: https://code.visualstudio.com/Docs/runtimes/unity

Start enable VSCode-Unity integragion:
    - Open Unity
    - Go to "Edit->Preferences->VSCode"
    - Check "Enable Integration"
    - Check "Use Unity Debugger"
    - Then click on "Write Workspace Settings"
    - Restart Unity

To enable debugging in VSCode:
    - Open VSCode 
    - Press [Ctrl+P]
    - Enter "ext install unity-debug"
    - Install extension

To start debugging press "F5" in VSCode and then start the scene in Unity
